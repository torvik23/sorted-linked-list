# SortedLinkedList

An automatically-sorted, type-safe linked list data structure for PHP

## Requirements

- PHP 8.1 or higher
- Composer for dependency management
- No runtime dependencies (zero-dependency library)

## Installation

```bash
composer require torvik23/sorted-linked-list
```

## Usage

See `simple-usage.php` script in `examples` folder.

## Testing

```bash
# Run all tests
composer test

# Run PHPStan analysis
composer phpstan

# Run PHPCS analysis
composer phpcs

# Run everything together
composer quality
```
